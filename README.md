## 📌 About the Project

AI Voice Coach helps job seekers practice interview answers using voice input and receive feedback on:

* Confidence
* Tone
* Clarity

It is specially designed for **first-generation college students and rural job seekers**.

---

## 🚀 Features

* 🎤 Voice-based interview practice
* 🤖 AI-powered feedback system
* 🔊 Professional model answers
* 📊 Confidence and tone analysis
